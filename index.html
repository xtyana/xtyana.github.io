How I got back on track

9 FEBRUARY 2019

So after a while I decided to start coding again; I tried to just continue from where I stopped but there was too much information that I had forgotten and I wouldn't be doing it properly.
So I had to go back to the basics and start from the beginning ; this time i went back to HTML;But instead of just trying to get through each stage I actually practiced coding carefully and cautiously so I retained what I was learning.
I finally got a copy for coding where i wrote down things I learned from the videos and I now have have own documentation.
Having my own documentation really helped me with coding because I'm able to put things in my own words and If I forget anything It's right there for me.
